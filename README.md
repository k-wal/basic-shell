# Shell

## Brief

Custom **shell** which has `cd`, `echo`, `pwd`, `ls`, `pinfo`, `remindme` and `clock`.

## To start the shell

- Compile:
	- `make`
- Run:
	- `./main`
- Remove:
    - `make clean`

## Commands and their Files

- **cd**: `cd.c`
- **echo**: `main.c`
- **pwd**: `main.c`
- **ls**: `ls.c`
- **pinfo**: `pinfo.c`
- **remindme**: `remindme.c`
- **clock**: `clock.c`
- **background (&)**: `system_commands.c`
